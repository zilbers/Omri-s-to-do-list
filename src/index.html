The perfect to-do list

Organize your day with this simple yet effective to-do list!

  • Add a new task by entering a task to the input bar bellow, choosing priority and then pressing add.
  • Sort the tasks by priority by pressing the sort button.
  • You can edit priority value by hovering over it and then pressing the Enter key.
  • Save your tasks by pressing the save button.
  • You can delete marked tasks by pressing the delete button.
